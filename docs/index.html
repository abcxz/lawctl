What does it actually do?

🔒

Protects your secrets

Blocks agents from reading or leaking .env files, SSH keys, API tokens, and credentials.

🛑

Stops dangerous commands

Catches rm -rf, curl|bash, and other commands that could wreck your machine.

📦

Controls git push

Asks permission before pushing code. No surprise deploys.

📄

Logs everything

Full audit trail of every action your agent took. Run lawctl log to review.

What it looks like

Claude tries to read your .env...
[lawctl] BLOCKED: write '.env' -- denied by policy
 
Claude tries rm -rf /...
[lawctl] BLOCKED: run_cmd 'rm -rf /' -- denied by policy
 
Claude edits src/app.ts...
[lawctl] allowed: write 'src/app.ts'
 
$ lawctl log
3 actions | 1 allowed | 2 blocked
